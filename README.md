
# 💎 Amnesia JSON Parser

> **Хватит тратить кристаллы Fan Stat!** 
> Зачем платить за аналитику, если можно выудить сообщения нужных пользователей из любого чата бесплатно и в пару кликов?

Этот скрипт позволяет вытащить историю сообщений конкретных людей из дампов Telegram и аккуратно разложить их по отдельным файлам.

---

## 📖 ИНСТРУКЦИЯ (Как это работает)

### Шаг 1. Экспорт данных 📥
1. Откройте **Telegram Desktop** и перейдите в нужный чат.
2. Нажмите на три точки в правом верхнем углу → **Экспортировать историю чата**.
3. **ВАЖНО:** В поле "Формат" обязательно выберите **JSON**!
4. Дождитесь, пока файл скачается **ДО КОНЦА**. 
   *⚠️ Если прервать загрузку, структура JSON нарушится, и скрипт выдаст ошибку.*

### Шаг 2. Запуск и парсинг 🚀
1. Запустите скрипт: `python zalua.py`.
2. Скопируйте путь к скачанному файлу `result.json` и вставьте его в программу.
3. Используйте интерактивное меню для настройки:

--- Меню управления ---
1️⃣ **Добавить еще чат** — если нужно собрать данные сразу из нескольких источников.
2️⃣ **Добавить пользователя/телей (ID)** — впишите ID тех, чьи сообщения ищем.
3️⃣ **Начать парсинг** — магия начинается здесь.

---

## 📄 Результат
После завершения в папке со скриптом появятся `.txt` файлы, названные по ID пользователей. Внутри всё разложено по полочкам:
- Кто написал
- В каком чате
- В какое время
- Текст сообщения
